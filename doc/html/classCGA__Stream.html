Detailed Description

CGA based output mechanism, similar to cout.

CGA_Stream enables the output of different data as string using the CGA screen. It is derived from the class CGA_Screen which makes additional formats and effects available. CGA_Stream extends the functionality of the classes CGA_Screen and O_Stream.
